## Environment

* Node.js >= 10
* Yarn >= 1.12

## Commands

Install with:

```shell
$ yarn
```

Build with:

```shell
$ yarn build
```

Start the application

```shell
$ yarn start
```

Test the application

```shell
$ yarn test
```
