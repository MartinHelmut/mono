# MonoRepo with Lerna

[![lerna](https://img.shields.io/badge/maintained%20with-lerna-cc00ff.svg)](https://lernajs.io/)

## Environment

* Node.js >= 10
* Yarn >= 1.12

## Commands

Install with:

```shell
$ yarn
```

Build with:

```shell
$ yarn build
```

Start the application

```shell
$ yarn start
```

Test the application

```shell
$ yarn test
```
